## Keywords

Keyword die wir noch nicht gelernt haben, diese für Anfänger fragen aufwerfen könnte.

| Keyword | Erklärung                                                                                                         | Url zu Docs |
|--------|-------------------------------------------------------------------------------------------------------------------|-------------|
| AsNoTracking() | Sagt Entity Framework: „Ich will die Daten nur lesen, nicht ändern“ → schneller.                                  | https://learn.microsoft.com/en-us/ef/core/querying/tracking |
| Trim() | Entfernt Leerzeichen am Anfang und Ende eines Textes.                                                             | https://learn.microsoft.com/en-us/dotnet/api/system.string.trim |
| Dependency Injection | Abhängigkeiten werden von außen bereitgestellt, statt sie selbst zu erstellen. Macht Code testbarer und sauberer. | https://learn.microsoft.com/en-us/dotnet/core/extensions/dependency-injection |
| [FromQuery] | Holt einen Wert aus der URL-Query (?name=xyz) und setzt ihn als Parameter.                                        | https://learn.microsoft.com/en-us/aspnet/core/mvc/models/model-binding |
| [ValidateAntiForgeryToken] | Schützt vor gefälschten Formularanfragen (CSRF-Angriffe).                                                         | https://learn.microsoft.com/en-us/aspnet/core/security/anti-request-forgery |
| FirstOrDefault() | Gibt das erste Element zurück oder `null`, wenn nichts gefunden wurde.                                            | https://learn.microsoft.com/en-us/dotnet/api/system.linq.enumerable.firstordefault |
| Any() | Prüft, ob mindestens ein Element existiert (true/false).                                                          | https://learn.microsoft.com/en-us/dotnet/api/system.linq.enumerable.any |
| CancellationToken | Ermöglicht das saubere Abbrechen von laufenden Tasks (z. B. bei Timeout oder Request-Abbruch).                    | https://learn.microsoft.com/en-us/dotnet/api/system.threading.cancellationtoken |
| async / await | Ermöglicht nicht-blockierenden Code → App bleibt schnell & reagiert weiter.                                       | https://learn.microsoft.com/en-us/dotnet/csharp/programming-guide/concepts/async |
